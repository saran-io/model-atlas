# AI Models Deep Dive — Content Strategy

This is Saran's 6-month AI models content project. One model per week.

## Structure
- `content/week-XX/` — each week's deliverables (blog, diagrams, code, social posts)
- `templates/` — reusable templates for consistency
- `assets/` — shared diagrams and images
- `ROADMAP.md` — full 24-week schedule with models and themes
- `CONTENT-PLAYBOOK.md` — the weekly routine and distribution checklist

## Conventions
- Blog posts are written in markdown, ready for personal site + Medium
- Diagrams use Mermaid (in `.mmd` files) for version control
- Code experiments are self-contained per week with their own README
- Social posts (LinkedIn, X) are pre-written in `social/` folder
